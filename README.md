# Pyodide Scraper on GitHub Pages

This folder is intended to be served by GitHub Pages (set Pages source to `/docs`).

- Open `index.html` from the GitHub Pages URL.
- Input the target URL and click **実行**.
- Click **CSVダウンロード** to download the generated CSV.

> Note: Accessing an `http://` URL from a GitHub Pages `https://` site may be blocked by browsers (mixed content / CORS).
